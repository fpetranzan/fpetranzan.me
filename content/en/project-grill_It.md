<p>"Grill It" is a Flutter mobile application designed to help users find ideal locations for organizing barbecues in Italy.</p>
<p>The app features an interactive map interface with advanced search and filtering capabilities to discover grill-friendly areas.</p>
<p>Users can locally save their favorite places, filter between free and paid locations, and view detailed information about available amenities such as parking, fountains, playgrounds, and restrooms.</p>
<p>The application uses geolocation services to find nearby spots and includes a reporting system for issues or suggestions of new locations.</p>
<p>It supports both Italian and English languages and integrates with Supabase for data management.</p>
<p>The interface is organized with a three-tab navigation structure: search with map, favorites, and settings.</p>
<p>The project aims to simplify outdoor event organization by providing accurate information about available barbecue locations.</p>
<p>It's designed for families and groups looking for suitable spaces for outdoor recreational activities throughout Italy.</p>
