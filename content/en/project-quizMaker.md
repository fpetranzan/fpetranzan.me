<p>Quiz Maker is a web application that allows users to create and take custom quizzes from JSON files.</p><p>The app features two main sections: a quiz player for taking randomized quizzes with immediate feedback, and an editor for creating, modifying, and managing questions.</p><p>It supports multiple JSON file uploads, question selection (5, 10, 20, or all questions), and provides search and filtering capabilities in the editor.</p><p>Built with standard web technologies (HTML5, CSS, Vanilla JavaScript), it requires no special installation.</p><p>The JSON format is structured with multiple-choice questions (4 options), correct answers, and detailed explanations.</p>
