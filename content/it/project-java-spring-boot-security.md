<p>Questo progetto rappresenta un sistema di sicurezza completo, sviluppato utilizzando il framework Spring Boot e JSON Web Tokens (JWT) per la gestione dell’autenticazione e dell’autorizzazione.</p><p>Tra le principali funzionalità implementate ci sono la registrazione degli utenti, la verifica delle email, il login, il recupero della password, l’autenticazione a due fattori e il logout.</p><p>Per la gestione dei dati ho utilizzato un database MySQL, mentre Spring Security è stato impiegato per implementare autorizzazioni basate su ruoli.</p><p>Le password sono protette attraverso l’algoritmo BCrypt, garantendo un alto livello di sicurezza.</p><p>Inoltre, ho personalizzato la gestione degli accessi non autorizzati per migliorare l'esperienza utente e rafforzare la protezione.</p>
