<p>Java Memory Game è il mio primo progetto complesso sviluppato durante il percorso di studi. Si tratta di una versione digitale del classico gioco 'Memory', implementata in Java 8 utilizzando Java Swing.</p><p>Il gioco include diverse modalità (singolo giocatore, due giocatori, contro il computer), livelli di difficoltà (4x4, 6x6, 8x8), e un sistema di salvataggio/ricarica delle partite.</p><p>Funzionalità aggiuntive come gestione dei turni a tempo e punteggio dinamico rendono l'esperienza più sfidante e coinvolgente.</p><p>Sebbene presenti alcuni bug e margini di miglioramento, questo progetto rappresenta un traguardo importante nel mio percorso di sviluppo software.</p>
